# Monthly Sales Analysis (Excel)

**Objective:**  
To analyze monthly sales and create a simple chart for visualization.

**Tools Used:**  
- Microsoft Excel

**Data Description:**  
The dataset contains 5 months of sales:
- January: 12,000
- February: 15,000
- March: 9,000
- April: 17,000
- May: 14,000

**Steps Performed:**  
1. Entered monthly sales data  
2. Created a column/line chart  
3. Added a title to the sheet  
4. Saved the file as monthly_sales_analysis.xlsx

**Insights:**  
- April had the highest sales (17,000)  
- March had the lowest sales (9,000)  
- Sales trend fluctuates month to month
